API

what is an API?

An application programming interface (API) is a system that enables programs to fetch data from a service automatically. APIs are useful for software developers to make applications that interface with another service.

what is this API?

The content API for this website is used internally to generate HTML pages using JSON data files. It can also be used by other software to dynamically fetch and display content from gabl.ink.
